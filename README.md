# Online Workshop Management Platform

Premium frontend-only workshop management platform built with React, JSX, and Vite.

## Tech Stack

- React
- React Router
- Framer Motion
- Lucide React
- Recharts
- React Hot Toast
- Axios placeholder client

## Run In VS Code

1. Open the project folder in VS Code.
2. Open a terminal in the project root.
3. Run `npm install`
4. Run `npm run dev`
5. Open the local Vite URL shown in the terminal.
